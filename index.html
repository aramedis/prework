HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The content is contained between an opening tag and a closing tag
  • An element refers to the entire element from opening tag to closing tag, whereas a tag refers only to what appears inside the angle brackets.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The padding property adds space around the content inside an element.
  • A margin property indicates how much space we want around the outside of an element.
  • More about the Box Model

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pull the latest main, important before creating branches or staging/commit
  • git add -A: stage the changes
  • git commit -m "insert notes on changes": commit the changes, -m flag is to associate a message with our commit

JavaScript

  • Control flow can be interrupted, however. We can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
  • Link .JS to HTML by inserting 
    <script src="./assets/script.js"></script> between footer and body

  • console.log(var); can be used to output variables to console for debugging.
  • A variable is a named container that allows us to store data in our code.
